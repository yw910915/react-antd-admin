## 安装依赖

```
yarn install
```

## 接口docker地址

```
https://gitee.com/hanyun_admin/react-admin-docker.git
```



## 功能说明

1. 登录认证
2. 仪表盘
3. 用户管理--用户列表、添加用户、编辑用户、删除用户
4. 角色管理--角色列表、添加角色、编辑角色、删除角色
5. 管理员管理--管理员列表、添加管理员、编辑管理员、删除管理员
